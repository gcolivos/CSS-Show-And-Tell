Differences between PX, EM, and REM

Why is this important?

When creating your CSS property dimensions, you can choose to format it in one of three ways:

  • "px" (for example, font-size=12px or padding:5px)
  • "rem" (for example, padding:2rem)
  • "em" (for example, padding:2em)

Using Pixels (px)

When you use pixels as a number, you are literally telling the DOM how many pixel lengths you require for the size or width of an element.

  • These are static numbers, they do not depend on anything else on the page.
  • There are a finite number of these units that will fit on any display, no matter how big or small.
This box has width:100px, height:100px, and margin:10px

Using REM (rem)

When you use rem as a number, you are telling the DOM what % of the font-size of the html you want.

  • This is relative, where 1 em = whatever the font size is, # of px.
  • For example, the font-size of the html of this page is 20px.
This box has width:10rem, height:10rem, and margin:1rem

Using EM (em)

When you use em as a number, you are telling the DOM what % of the font-size of the CURRENT ELEMENT you want.

  • This is relative, where 1 em = whatever the font size is of that element, # of px.
  • For example, the font-size of THIS PARTICULAR DIV of this page is 40px.
width:5em, height:5em, margin:0.5em

So, one more time...

Width:200px
Width: 10rem
Width: 5em

And that is how px, rem, and em work for widths. The end.
